       - 在函数开始时，如果 context_index 不为 None，先为内容中所有候选字符串的位置收集上下文信息
       - 修改正则提取逻辑：使用 re.finditer 替代 re.findall 来获取位置信息

         ```python
         candidate_matches = list(re.finditer(r'"([A-Z][A-Za-z][^"]{4,120})"', content))
         ```

         ```python
         positions = [m.start() for m in candidate_matches if m.group(1) == s]
         if context_index is not None:
             all_tags = set()
             for pos in positions:
                 tags = detect_context(context_index, pos)
                 all_tags.update(tags)
             ctx = sorted(all_tags) if all_tags else ["default"]
         else:
             ctx = []
         ```
       - 在结果 dict 中添加 "contexts": ctx
