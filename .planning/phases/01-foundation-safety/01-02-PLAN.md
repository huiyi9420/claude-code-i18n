    2. 实现 scripts/i18n/io/file_io.py -- 两个函数:
       a) atomic_write_text(target: Path, content: str, encoding: str = 'utf-8') -> None:
          - tempfile.mkstemp(dir=target.parent, prefix='.tmp_', suffix=target.suffix)
          - os.fdopen(fd, 'w', encoding=encoding) 写入 content
          - os.replace(str(tmp), str(target)) 原子替换
          - except BaseException 时清理临时文件并 re-raise

       b) atomic_copy(src: Path, dst: Path) -> None:
          - tempfile.mkstemp(dir=dst.parent, prefix='.tmp_', suffix=dst.suffix)
          - os.close(fd) 关闭文件描述符
          - shutil.copy2(src, tmp) 复制内容
          - os.replace(str(tmp), str(dst)) 原子替换
          - except BaseException 时清理并 re-raise

    - 使用 os.replace() 而非 os.rename()，跨平台更安全
    - 异常处理用 BaseException 而非 Exception，确保 KeyboardInterrupt 等也清理
